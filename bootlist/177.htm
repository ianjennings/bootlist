2) Where's My XP Clipboard?

Bruce asks - Am I supposed to have Clipboard in XP Home? Can't find it in Start | Programs | Accessories. Checked in Add/Remove Windows Components and it's not there either.

Carey F. responds:
Perform the following:

1. Right-click on your Desktop and select: NEW | SHORTCUT
2. In the Create Shortcut window, type in: CLIPBRD.EXE , and click Next
3. In the next window, type in: ClipBoard Viewer , and click Finish
4. A ClipBoard Viewer shortcut icon will appear on your Desktop.

4) 4.3 Gigs Of Temp Files [Continued]

John comments regarding [4.3 Gigs Of Temp Files] from issue 175

A much simpler solution is to:

1. Just restart the PC.
2. Then open Windows Explorer, navigate to the temp folder, then click on the Date column, sort everything descending by date. The delete everything older than today (select the first one you want to delete, then shift select the last to select everything in between, then shift/delete to permanently delete them all.

5) What's My IP Address? [Continued]

Aristiana comments regarding [What's My IP Address?] from Issue 174

In Windows XP, if using the PPPoE connection, you can get the IP address somewhere else, which is also a lot easier.

Just either double-click on the two little screens on the tray and click on the Support tab, or go to Start -> Control Panel -> Network connections -> double-click on the DSL connection.

Same thing can apply if the DSL uses DHCP. Though the screens may not be in the tray, but the good thing is that on the Support tab there's a Repair button which does exactly the same as ipconfig /renew.

Some images that shows what I' talking about. It's only DHCP, but the look is the same for PPPoE:
http://support.shaw.ca/troubleshooting/dhcpxp.htm

Though they circled the Properties button for unknown reason. It's really the Support tab. All of this is not available in Windows 2k.

6) An Easy Way To Get Rid Of Added IE Toolbars

A customer just showed me this. Right mouse click in a blank space on the IE Toolbar, usually on the right side and deselect any toolbars displayed there. Often many of the third party add-ons are there. This wont work on all added toolbars but it will work with many of them.
