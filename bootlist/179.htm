5) NTLDR Is Missing

jd wrote - [My pc says] missing NTLDR while trying to load windxp home.

t.cruise suggests:

http://www.computerhope.com/issues/ch000465.htm

Rick R. says:
Boot the WinXP CD, load the Recovery Console:

HOW TO: Install and Use the Recovery Console in Windows XP [Q307654]:
http://support.microsoft.com/?kbid=307654

From the command prompt, copy ntldr from the I386 folder on the CD to the boot partition.

C:\>copy X:\I386\ntldr C:\

Where X is whatever letter is assigned your CD drive.
