2) Is The ProductKey On The CD?

Alex questions - I was reformatting my hard drive, and apparently my XP CD is scratched, can I use my product key with another CD without violating copyright laws or anything like that?

Alex says:
As long as both versions are identical it will work. The Product Key is not tied to the CD.

Dale R. adds:
You can use your product key with another CD as long as it contains the same version of XP as the CD that came with your product key, e.g. an OEM key can only be used with an OEM disc etc.


*** It's a common misconception that the ProductKey is on the CD. One has to remember tho that not all types of XP cds are the same. For example, if you bought your copy of XP in a store that came in a box [Retail CD] and lost it, your key will not work with your friends Gateway XP restore cd or an OEM CD.

5) New XP SP2 IE Tweaks

Steve poses - Using XP Pro w/ SP2 gives me this security warning when visiting certain sites, for example the New York Times:

File Download - Security Warning - Do you want to open or save this file?

The files in question are JScripts with .js endings. Is there a way to suppress these warnings through settings in IE 6.02?

Carey F. explains:
Launch Internet Explorer and from the toolbar select: Tools | Pop-up Blocker | Pop-up Blocker Settings and select your desired notification level settings.

How to configure the Pop-up Blocker in Internet Explorer:
http://support.microsoft.com/?kbid=843016

Description of the Internet Explorer Information Bar in Windows XP SP2:
http://support.microsoft.com/?kbid=843017
