1) MS Says My Key Is Invalid When Installing SP1

mandie posts - I downloaded the sp1. It did it fine apparently. But when it came to the next step it said the key was wrong.

Carey F. replies:
Run the Windows XP Product Key Validation Assistant:
http://www.microsoft.com/resources/howtotell/ww/windows/default.mspx

If you are getting the following error message when attempting to update to SP1:

Error Message: The Product Key Used to Install Windows Is Invalid:
http://support.microsoft.com/?kbid=326904

You'll have to purchase a new Full Version of Windows XP Professional and perform a Repair Install using the new Windows XP Pro CD and Product Key that comes with it. (You cannot simply change the Product Key)

How to Perform a Windows XP Repair Install:
http://www.michaelstevenstech.com/XPrepairinstall.htm

Reporting Piracy:
http://www.microsoft.com/piracy/Reporting.mspx

2) Customize Windows Update

Martin R. asks - I have a little problem with WindowsUpdate, everytime that I go check for updates I [have] NVidia driver (I have a motherboard with Nvidia chipset, and a TI4400 videocard), but I have more recent drivers installed. How come windowsupdate still list those outdated driver, if I have already installed more recent, directly from the Nvidia web site.

OR how can I configure the Windows Update to not show me these updates?

Wesley V. answers:
Go to Windows Update | In the left head frame | Under: Other Options | Click: Personalize Windows Update | Click: Scan for updates | Pick from the following items to specify what you see when you use Windows Update. To apply your changes, click Save Settings. Choose which categories and updates to display on Windows Update.
