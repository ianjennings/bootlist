2) NTLDR Is Missing

Seeker wrote - [I keep getting a message that] NTLDR is missing and to press CTRL+ALT+DEL to restart which I have but I keep coming back to the same thing again that NTLDR is missing. How can I please restore my operating system back up again without having to reformat?

Rock points to:

http://support.microsoft.com/?kbid=320397


*** It seems to be the same old thing ie too many files in the root directory ie C:\ can and historically even in previous versions of windows can and/or may cause problems. What often happen is folks install programs to C:\ instead of going with the recommend folder in Program Files or, they just copy too many files to C:\ to make them easy to find. Or, they unpack archives to C:\.

Looking at the root directory is one thing I've learned to do when faced with odd windows problems, especially on older win9x PCs. You really want to minimize the number of non-MS files there. I did just cover NTLDR last month but the MS link regarding too many files in Root is new news to me so it may help.

Here's the previous link/info anyway:

http://www.bootdisk.com/bootlist/179.htm#5
