1) Hide Your Webpage Email Address From Spammers

Gerry P. emailed - The reason I'm writing is to suggest a different (perhaps more secure) way to hide your e-mail address [on your webpages] from spammers. My script [also] allows additional elements like default Subject, CC, Bcc, Body, etc.


*** Hi Gerry, I'm already using a script which works:

http://www.javascript-page.com/emailprotect.html

Gerry mails back: I know it works. I've used it :) What my script offers that's new and (imho) better is:

1) the string "mailto:" is not there
2) the "@" sign is not there
3) a "tail" for default subject, body, cc, etc.

In my opinion, it's an improvement not to have "mailto" in the source text, because if I were designing a filter or tool to harvest addresses, I would have it search for "mailto:" and "@", which would get a hit on your pages, but not on my pages.


*** I put together a few pages to make all this easier to view, copy, paste, and setup.

http://www.bootdisk.com/txtfiles/email.txt

I like the fact that you can use Gerry's script to put in a Subject which one can then use to filter mail that doesn't contain the subject if one so desires.

If you're wondering, "why bother" with all this? Well, email addresses on webpages using standard html tags created by your html editor are harvested by "spiders" and are accumulated into spam lists. The two best ways to combat that and minimize spam due to webpage email harvesters are to either create a small .gif with your email address as the graphic or use one of the scripts discussed above.

3) Ten Steps To XP Fresh

hajacan wrote - I would like to reformat my computer. I already have nfts and just want to reformat then install win xp pro. What are the command prompts?

Carey F. answers:
The Windows XP CD is bootable and contains all the tools necessary to partition and format your drive. Follow this procedure and allow Windows XP to partition and format your drive:

NOTE: It would be best to physically disconnect all your peripheral hardware devices, except the monitor, mouse and keyboard, before installing XP.

NOTE: If you have an internal Zip Drive installed, physically disconnect the EIDE and power cable to it before proceeding, otherwise your main hard drive may not be assigned the customary C: drive letter. After installing Windows XP, you may then reconnect it.

1. Open your BIOS and set your "CD Drive as the first bootable device".

Accessing Motherboard BIOS:
http://www.michaelstevenstech.com/bios_manufacturer.htm

2. Insert your Windows XP CD in the CD Drive and reboot your computer.

3. You'll see a message to boot to the CD. Follow the instructions.

4. The setup menu will appear and you should elect to delete all the existing Windows partitions, then create a new partition, then format the primary partition (preferably NTFS) and proceed to install Windows XP.

5. Clean Install Windows XP:
http://michaelstevenstech.com/cleanxpinstall.html

6. Immediately after installing Windows XP, turn on XP's Firewall:
http://www.microsoft.com/athome/security/protect/default.mspx

7. After Windows XP is installed, visit the Windows Update website and download the available "Critical Updates". [Or install SP2]

8. After installing the critical updates, be sure and visit the support website of the manufacturer of the computer to download and install any available Windows XP compatible drivers, such as video adapter and audio drivers.

9. If you happen to run into any installation difficulties, use the following resource:

Troubleshooting Windows XP Setup:
http://www.kellys-korner-xp.com/xp_setup.htm

10. Get Windows XP Service Pack 2 with Advanced Security Technologies:
http://www.microsoft.com/athome/security/protect/windowsxp/choose.mspx


*** Yes there are other methods to install XP fresh, but Carey put some time into this list and it's fine, and may even have a few tips you weren't aware of before.

5) Using W2K Drivers For HP Printer On XP

Mr. Gabay asked - I have XP and a older HP printer. XP installed its driver for the printer but I'm missing all the bells and whisles that came with the printer on my Windows 98 PC. On the HP website there is no XP driver for download but there is a Windows 2000 driver. Can I use that to get all the special features I want like head allignment and large page printing?


*** Well, in all honesty that is a legitimate thing to try, and many techs would take a shot at it and even recommend that you try it yourself. After all, Windows 2000 is just NT5 and XP is just NT5.1 so it "may" work.

But, it's a risk and you better do all the standard precautions first, like making sure you have a good restore point, a separate copy of the good, working registry, closing down or disabling all background apps first, removing the old driver, then following the printer install docs EXACTLY.

6) Customize The Places Bar In The Open/Save As Boxes

ubpc wrote - How can I add a folder ("C:\this folder") to the list of places displayed on the left side bar of the "save as" window--- currently it lists:

"My Recent Documents"
"Desktop"
"My Documents"
"My Computer" and
"My Network Places"

Anando answered:
This can be easily accomplished with the help of TweakUI powertoy for Windows XP. Under TweakUI, navigate to 'common dialog | places bar' and you can specify your custom places here.

Download TweakUI here:
http://www.microsoft.com/windowsxp/pro/downloads/powertoys.asp

Rock adds:
In addition to what Anando said you can only have five entries there, so you'll need to replace one of them.
