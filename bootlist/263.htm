1) CDrom Drives Seen, But Not Working

Lando C. wrote - I am having problems with my cd-rom drive and dvd drive. I can see both drive in my windows explorer. But when I insert a cd-rom I can not read it. I removed the cd-rom drive and dvd -drive from the device manager. After restarting Windows XP it automatically detected both drives. But still I cannot read cd and dvd. I looked also in the BIOS, both drives are mentioned in the bios.


*** Some things to check:

1. A drive or both drives may not work on the same channel if the other drive on the same channel is bad.
2. Could be a jumper setting problem if you've recently installed a drive incorrectly. eg if you add a Slave drive to a channel and set the Jumper to Slave you also may have to change the original drive jumper to Master.
3. An IDE cable may be bad and/or shorting out on a sharp edge of the innards of the case.
4. An IDE cable may have become partially mis-seated from the motherboard or drive.
5. You may have a boot sector virus.
6. Boot do DOS with a bootdisk that sees cdrom drives and see if you can read a disk. This will tell you if it's a Windows problem or a hardware problem. Note that just doing a Dir command will only read the FAT so you also have to see if one can open Edit or a text file, etc. after a dos boot.
7. Worst case is that the channel the CDrom drives are on is not working correctly or is bad, tho very rare especially since XP is detecting the drives.

The above list is not at all complete for cdrom drive problems. However, the first step is seeing if your problem is hardware or software based. That's one good reason bootdisks and bootable CDs that take you to DOS are still very valuable as they dont care what OS you have installed. If it doesn't work in DOS, then it wont work in Windows. And if it doesn't work in DOS, then you have a hardware problem, not a Windows problem.
