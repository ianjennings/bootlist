1) Should I Disable BIOS Virus Detection When Installing XP?

Samantha asks - I'm having problems loading XP onto my new pc I just built. It seems to get stuck after the first reboot. My boyfriend told me I should disable the virus detection option in my bios. Is this safe?


*** Yes it's safe. Your bios cannot detect a virus per se. The function you are referring to is just a [boot sector write warning]. ie it's a function that is designed to warn you if something is about to write to the boot sector. Which does include, old school boot sector viruses in addition to the files your pc needs to see when starting up.

Most of the time you get a chance to say Yes or No. But with OS installs this _prompt_ may NOT occur or you may just not see it. So, yes, disable it when doing OS fresh installs and OS upgrades so that the new OS can write the files it needs to the boot sector. You can re-enable it later on.

In other words, when you turn on your PC it fist looks at the boot sector of the hard drive or floppy drive for the basic startup files. io.sys is the classic example of a file that resides in the boot sector. This tells your pc how to use your hard drive.

3) How To Disable XP Asking For Boot Up Password

Glen posts - Just lately the window asking for a password shows up on boot up. How do I get it back to just going into windows without that screen showing up?

Phil answers:
Auto Logon:

Start, run, type in: control userpasswords2, hit enter.

Uncheck: users must enter a user name and password to use this computer.

Xp will then ask you to enter the user and password you want to auto-logon to. Or, download and install tweakui for xp and use its auto-logon feature.

http://www.microsoft.com/windowsxp/pro/downloads/powertoys.asp

4) How To Preview AVI File Downloads

burp inquires - How do I preview AVI files I'm downloading from newsgroups without having to DL the entire movie? I have XP Pro and WMP 9.0 and a bunch of codecs.

Unk answers:
Download the free program, AVIPreview.exe:
http://www.avipreview.com/avipreview.htm

The program doesn't need to be installed, just unzipped. Now place a shortcut to it in your Send To folder. Usually, movies from newsgroups are uploaded as split parts. Just download the 1st split part and save it to a temporary folder.

If it was .avi.001, rename it .avi If it was .r00 or part01.rar, then double-click it, Extract To, and check the box, Keep broken files & save to the temporary folder. If it was .avi (01/50), then right-click it and select Split sections, then download just that 1st piece & save to the temporary folder.

Once you have the 1st piece of the avi file, right-click it and use the Send To option to view it with AVIPreview.exe. Sometimes, you need to download several of the beginning sections to have enough of the movie to know what it's about.
