2) Fastest Way To Get To/Modify A Registry Key

Chuck wrote - If I'm reading thru Microsoft (or other) documents, and find a registry reference, such as in the [following] example [of a specific key I was told to change], what's the fastest way of examining that entry in my registry?

[In other words, does] anybody know of a utility that lets me copy and paste (or dream on - right click?) a key such as "HKEY_LOCAL_MACHINE\System\CurrentControlSet\Services\Netbt\Parameters\", and open the registry key easily?

The only way I know is to:
1) Start Regedit
2) Open HKLM
3) Open System
4) Open CurrentControlSet
5) Open Services
6) Open Netbt
7) Open Parameters

What a pain.

### Begin Example [This is where an MS KB article asks me to change a value for a key. If you have to change multiple keys in different areas it can be quite annoying]

RESOLUTION
WARNING: If you use Registry Editor incorrectly, you may cause serious problems that may require you to reinstall your operating system. Microsoft cannot guarantee that you can solve problems that result from using Registry Editor incorrectly. Use Registry Editor at your own risk.

To resolve this issue, use Registry Editor (Regedit.exe) to locate the following key in the registry:

HKEY_LOCAL_MACHINE\System\CurrentControlSet\Services\Netbt\Parameters\ Change the value of the EnableProxy value in the preceding registry key to 0 or 1, quit Registry Editor, and then restart your computer.

### End Example

Ramesh answers:
Try RegJump from Sysinternals.com:
http://www.sysinternals.com/ntw2k/source/misc.shtml#regjump

http://www.sysinternals.com/files/regjump.zip

Usage Example - Click Start/Run and type this:

C:\Regjump "HKEY_LOCAL_MACHINE\System\CurrentControlSet\Services"

Assuming regjump.exe file is placed in C:\


*** Ramesh also said a program called 12ghosts had a function that does the same with a nice interface. Yes, I tried 12ghosts yet again, and yes, it was fun and worked but it was just a tad too intrusive for me to actually post the link :) Easy to find tho using google if you want to try it out. At least it uninstalled OK.

3) Bypass Username/Password Prompt

Grimbo inquired - Using XP Pro. How do I reset my system so that it loads everything up without asking for username and password?

mark answered:
Click Start, Run and type "control userpasswords2", and click Ok. Uncheck "Users must enter a user name and password to use this computer" option, and click Ok. Type-in the username and credentials.

5) Fastest Way To Backup The Registry

KickingFox asked - For Windows 98 and Windows XP what's the quickest way to make backups of the registry without typing any dos commands or installing a third party utility?

Ron M. answers: To create a registry backup in Windows 98/98SE/Me use Start - Run and enter the following command:

scanregw /backup

In Windows XP you need to create a manual restore point using Start | All Programs | Accessories | System Tools | System Restore.


*** In case you've forgotton how to go back to a previous win9X registry boot to dos command prompt only and type:

scanreg /restore

Then choose the date [restore point] you want to go back to.
